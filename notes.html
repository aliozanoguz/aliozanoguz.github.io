Big İdea 4

The Internet ✶ The internet is a worldwide network that uses common protocols to link computers. ✮ Information is sent in discrete units known as packets, which can follow distinct routes and move independently. ✶ By detecting mistakes and making sure packets arrive in the right order, the Transmission Control Protocol (TCP) guarantees dependable data transfer. ✶ The Internet Protocol (IP) transports data between devices and gives each one a unique address. ✮ Human-readable domain names, like google.com, are converted into IP addresses via the Domain Name System (DNS). Tolerance for Failure ✶ The capacity of a system to function even when some of its components fail is known as fault tolerance. ✶ The internet is built to withstand failures by dynamically rerouting data via alternate routes in the event that a connection breaks. ✶ Network infrastructure redundancy reduces the effect of outages and increases dependability. Parallel and Distributed Computing ✶ To increase efficiency, parallel computing splits a workload into smaller subtasks that execute concurrently on several processors. ✶ In distributed computing, several computers collaborate to solve an issue, frequently over a network.
